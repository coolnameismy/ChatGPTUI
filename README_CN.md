### 功能特点
ChatGPTUI 具有以下功能：

🚀 对话历史记录
💬 用户信息
🎉 模型选择
💎 用户输入和 GPT 响应
🚦 开启新对话

### 安装步骤
要在本地运行 ChatGPTUI，请按照以下步骤进行：

```bash
# 克隆代码仓库
git clone https://github.com/coolnameismy/ChatGPTUI.git

# 进入项目目录
cd ChatGPTUI

# 安装依赖
npm install // 或者 yarn add

# 启动应用
npm start // 或者 yarn start
```

在浏览器中打开 http://localhost:3000 即可查看应用。
🎉 尽情享受 ChatGPT 类似的体验吧！
